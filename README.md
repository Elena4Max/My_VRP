# Task for Junior VRP Engineer Position, VanOnGo

## Зависимости

Jsoncpp

* `sudo apt-get install libjsoncpp-dev`
* `sudo ln -s /usr/include/jsoncpp/json/ /usr/include/json`

## How to use

* `git clone https://github.com/Elena4Max/My_VRP.git`
* `cd Out/`
* `cmake ../`
* `make`
* `./main`

## Бонус №1

* `cmake -DBONUS=bonus1 ../`
* `make`
* `./main`
